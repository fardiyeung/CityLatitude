Summary: Latitude vs. Max Temperature

The purpose of the project is to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities. After assembling the dataset. we use Matplotlib to plot various aspects of the weather vs latitude. Factors we looked at included: temperature, cloudiness, wind speed, and humidity . This site provides the source data and visualization created as part of this analysis, as well explanations and descriptions of any trends and correlations witnessed.
